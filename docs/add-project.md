# 如何添加一个新作品

现在项目已经有真实管理后台，后续新增作品不需要再改代码。

## 后台入口

打开：

```text
http://127.0.0.1:5174/#admin
```

部署后把域名换成你的线上域名，例如：

```text
https://your-domain.vercel.app/#admin
```

## 填写建议

- `Slug`: 英文短链接，只用小写字母、数字和中横线，例如 `ai-task-planner`。
- `标题`: 让 HR 能一眼知道项目是什么，不要写成内部代号。
- `简介`: 一句话说明项目解决的问题和结果。
- `问题`: 写为什么要做，而不是只写用了什么技术。
- `方案`: 写你如何拆解、设计和实现。
- `成果`: 写当前可看到的结果，优先写 Demo、截图、流程或数据。
- `技术栈`: 写你能讲清楚、面试能展开的技术。
- `Demo 链接`: 有可访问页面就填写。
- `源码链接`: GitHub 仓库公开后填写。

## 发布规则

- 勾选“发布到公开作品集”后，访问者才能看到这个项目。
- 勾选“设为重点项目”后，它会进入首页重点展示区域。
- 暂时不想展示时，取消发布即可，不需要删除。
